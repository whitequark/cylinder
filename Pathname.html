An empty path.

is_empty p returns true if p is an empty path.

extend p p' appends path p' (represented as string) to path p.

to_list p returns list of components of p.
val filename : t -> string
filename p returns the last component of p. If p is empty, raises Invalid_argument.
val basename : t -> t
basename p returns all but the last component of p. If p is empty, raises Invalid_argument.
val parent : t -> t
parent p returns all but the last component of p. If p is empty, returns p.
val extension : t -> string
extension p returns the part of the last component of p after the last dot.
val no_extension : t -> string
no_extension p returns the part of the last component of p before the last dot.
