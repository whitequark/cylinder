type t
The type of storage servers.
val create : Backend.t -> [ `Router ] ZMQ.Socket.t -> t
create backend sock creates a server for backend backend and a ZeroMQ socket sock. The maximum message length of sock must be set to max_message_size.
val to_socket : t -> [ `Router ] ZMQ.Socket.t
to_socket server returns the ZeroMQ socket used by server.
val listen : t -> unit Lwt.t
listen server starts and returns a thread that handles requests to server. The thread will terminate on any unexpected failure.
