Module Block.Client

module Client: sig .. end
type t
The type of storage clients.
val create : [ `Req ] ZMQ.Socket.t -> t
create sock creates a client for a ZeroMQ socket sock. The maximum message length of sock must be set to max_message_size.
val to_socket : t -> [ `Req ] ZMQ.Socket.t
to_socket client returns the ZeroMQ socket used by client.
val get : t ->
       Block.digest ->
       [ `Malformed | `Not_found | `Ok of string | `Unavailable ] Lwt.t
See Block.BACKEND.get. If the returned data doesn't match the digest, returns `Malformed.
val exists : t -> Block.digest -> [ `Not_found | `Ok | `Unavailable ] Lwt.t
See Block.BACKEND.exists.
val put : t ->
       Block.digest_kind -> string -> [ `Not_supported | `Ok | `Unavailable ] Lwt.t
See Block.BACKEND.put.
val erase : t -> Block.digest -> [ `Forbidden | `Ok ] Lwt.t
See Block.BACKEND.erase. If this client is not authorized to perform erase operations, returns `Forbidden.
val enumerate : t ->
       string ->
       [ `Exhausted | `Forbidden | `Ok of string * Block.digest list ] Lwt.t
See Block.BACKEND.enumerate. If this client is not authorized to perform enumerate operations, returns `Forbidden.
val digests : t -> [> `Forbidden | `Ok of Block.digest Lwt_stream.t ] Lwt.t
digests client returns a stream of all digests, collected through enumerate. If this client is not authorized to perform enumerate operations, returns `Forbidden.
